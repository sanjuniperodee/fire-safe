# CRM DEPLOY

## Project setup
```
make install
```

### Run migrations
```
make migrate
```

### Make migrations
```
make makemigrations
```

### Start crm
```
make start
```

### Stop crm
```
make stop
```

### Backup database
```
make backup
```

### Restore backup
```
make restore
```

### Go to bash
```
make shell
```

### Go to django shell
```
make django_shell
```

### Remove containers
```
make destroy
```

### Collect static
```
make collectstatic
```
